# todoapp_project
Production‑grade modular Terraform for Azure with DevSecOps scanning and multi‑env Modular Terraform project deploying Azure infrastructure for Todo App. Includes prod/dev environments with RG, VNet, subnets, NSG, NIC, and two VMs (frontend &amp; backend). Secured with TFLint, tfsec, Checkov, and Terratest via Azure DevOps pipeline.
